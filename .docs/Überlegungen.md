# Counter-App

Mehrere Views:

- main View 
- Listen View


---

### main View:

auf dem main View soll man den Namen der Liste eingeben können und oben auf ein + drücken können, die Liste wird dann unter dem Namen gespeichert

---

### Listen View:

wenn man die Liste mit dem Namen gespeichert hat, kann man nochmals darauf klicken und es wird der View der Liste geöffnet, in dieser hat man dann drei button einen für Plus, einen für Minus und einen für reset, der dann das geschpeicherte zurücksetzt (außer dem Namen).

außerdem soll man sehen wenn man die Liste zum letzten mal aktualisiert hat.

# Counter-App

Mehrere Views:

- main View 
- Counter View


---

### main View:

auf dem main View soll man den Namen des Counters eingeben können und oben auf ein + drücken können, Counter wird dann unter dem Namen gespeichert

---

### Counter View:

wenn man den Counter mit dem Namen gespeichert hat, kann man nochmals darauf klicken und es wird der View des Counters geöffnet, in dieser hat man dann drei button einen für Plus, einen für Minus und einen für reset, der dann das geschpeicherte zurücksetzt (außer dem Namen).

außerdem soll man sehen wenn man die Liste zum letzten mal aktualisiert hat.

# Design-Überlegungen

- es soll auf dem `main-view` die überschrift, das plus, und die eingabe des namen und der footer fix sein und nur der teil mit den counter-listen wenn sie mehrere werden zum scollen gehen.
- im `counter-view` soll der footer auch fix sein und oben der abstzand erhöht werden
- `Einstellungsview` - muss erst erstellt werden, in dem man fom hellen design zum dunkeln wechseln kann und einstellen kann wann die Counter auto resetet werden - standart ist sie werden manuell resettet
